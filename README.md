# CSV

一个基于大模型的 CSV 数据分析与可视化工具。  
用户上传 CSV 文件后，可以通过自然语言直接提问，系统能够自动完成数据分析、信息提取和图表展示。

---

## ✨ 项目简介

本项目是一个使用 **Streamlit** 构建的智能数据分析 Web 应用。  
用户无需手写 Pandas 代码，只需要上传 CSV 文件并输入问题，例如：

- “销量最高的产品是什么？”
- “请帮我统计每个月的销售额”
- “生成一个条形图展示不同地区订单数量”
- “找出评分最高的前10条记录”

系统会自动理解问题，并返回对应的：

- 文字答案
- 表格结果
- 条形图
- 折线图
- 散点图

---

## 🚀 功能特点

- 支持上传 **CSV 数据文件**
- 支持自然语言数据分析
- 支持返回 **文字结论**
- 支持返回 **结构化表格**
- 支持生成 **条形图 / 折线图 / 散点图**
- 支持查看上传后的 **原始数据**
- 页面简洁，适合演示、学习和项目展示

---

## 🛠 技术栈

- **前端 / 应用框架**：Streamlit
- **数据处理**：Pandas
- **大模型调用**：OpenAI
- **智能分析代理**：LangChain Pandas DataFrame Agent
- **输出格式**：JSON结构化返回
